Widget that shows the status of the PlayStation Network on your blog.

== Description ==

This sidebar widget displays the status of the PlayStation Network on your blog. It's simple to install and no configuration is needed.

== Installation ==

1. Upload `psn-status-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your Sidebar and you're done!
